Para crear un repositorio nuevo en github
1. git init
2. git add .
3. git commit -m 'Prime commit'
4. git branch -M main
5. git remote add origin https://github.com/tuusuario/turepositorio.git
6. git push -u origin main

Para subir un commit a su repositorio.
1. git add .
2. git status
3. git commit -m 'Segundo commit'
4. git push -u origin main
